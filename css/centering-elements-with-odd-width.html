Resize your browser width slowly too see how the centered div has wonky positioning because it is an odd number of pixels wide.

Wonkyness while resizing the browser is not the problem though; it is that you don't know how wide your users browser is so you don't know if the centered element is 1px to the left or right at the center, which can cause some pixel perfect designs to look ugly.
