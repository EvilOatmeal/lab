No ID selectors

So you have this widget that you want to place anywhere on your page, but to do this you need to keep your selectors at the same specificity, or else you will have some rules whose properties will always overwrite those of rules with selectors that have lower specificity, and it won't be possible to place your widget wherever you want to.

Here is the widget we will use. It only uses class selectors.

I'm a modular widget. Place me anywhere!

Here we place the widget inside a box that uses an ID selector, which has higher specificity than the widget and causes some of the widgets properties to always be overwritten.

This box uses an ID selector.
