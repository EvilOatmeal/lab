Rhythmic spacing scale

When setting whitespace and line-heights a spacing rhythm is defined on a scale around a base number (magical number), and this rhythmic scale is then used consistently throughout the design.

It's nice too set the default whitespace and line-height values of off key steps in the rhythmic scale for a pleasing vertical rhythm.

Nice reading about rhythm and "the magical number/scale"
