-webkit-text-size-adjust: none disables text zoom in desktop browsers

-webkit-text-size-adjust is used to prevent iPhone and iPads from resizing the text when you change the devices orientation. But when using the value of none it will also disable the ability to zoom the text on desktop browsers. This doesn't happen with the value 100%, and it still fixes the text scaling on orientation change problem.

So use -webkit-text-size-adjust: 100%
