Page gutters

It's important to use page gutters to let the content breath so it's easier to read.

Pages with a max width usually have nice gutters, but when the window is too narrow or you zoom the page these gutters might dissapear completely and the content will touch the edge of the window, making the text hader to read.

To fix this you can add horizontal margin or padding to a wrapping element of the page, then when the window is narrower than the pages max width this padding will make sure you still have a gutter.

On this page, if your browser window is wide enough you'll see everything in a nice centered column with gutters on both sides. But if you make the window narrower you'll see that section 1 keeps a gutter around it while section 2 does not.
