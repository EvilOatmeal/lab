When using no semicolons fails

If you have the return value on a new line

This: 
return
    true
Is interpreted as: 
return;
    true;
Fix it by writing it all on one line instead: 
return true

If a line starts with parenthesis

This: 
a = b + c
(d + e).print()
Is interpreted as: 
a = b + c(d + e).print()

The ugly fix, unless you want to add semicolons to the end of lines, is to add a semicolon to the beginning of every line that begins with a parenthesis.

a = b + c
;(d + e).print()
